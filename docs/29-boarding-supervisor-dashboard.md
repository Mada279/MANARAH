# 29 — لوحة المشرف التربوي للداخلي

## الهدف

إضافة لوحة عملية للمشرف التربوي في داخلي البنين بدور التحفيظ، تعرض حالة اليوم وتساعده على معرفة من يحتاج متابعة مباشرة.

## Endpoint جديد

```http
GET /api/v1/organizations/:organizationId/education/boarding/supervisor-dashboard
```

ويمكن التصفية باسم المشرف أو التاريخ:

```http
GET /api/v1/organizations/:organizationId/education/boarding/supervisor-dashboard?supervisorName=...&date=2026-07-10
```

## ماذا تعرض؟

- طلاب المشرف.
- متابعات اليوم.
- الطلاب الذين لم تُسجل لهم متابعة اليوم.
- من لم يحضر صلاة الفجر.
- من لم يحضر الحلقة.
- الحالات الصحية أو الإشرافية.
- الطلاب منخفضو السلوك.
- الطلاب منخفضو النظافة.
- الطلاب منخفضو الانضباط في النوم.
- تنبيهات اليوم.
- توصيات سريعة للمشرف.

## تحديث Dashboard

تمت إضافة تبويب جديد:

```txt
لوحة المشرف
```

يعرض:

- مؤشرات اليوم.
- تنبيهات اليوم.
- توصيات سريعة.
- الطلاب بدون متابعة اليوم.
- قوائم المتابعة السريعة.
- زر ينقل المشرف إلى تبويب داخلي البنين لإضافة متابعة يومية.

## تحديث Smoke Test

تم تحديث الاختبار ليتحقق من:

```txt
GET /education/boarding/supervisor-dashboard
```

ويتأكد من وجود قائمة تنبيهات.

## المرحلة التالية

- تحديد مشرف لكل مستخدم فعلي.
- عرض الطلاب حسب المشرف الحالي فقط.
- إضافة إشعارات يومية تلقائية.
- إضافة قائمة مهام للمشرف.
- تحويل تنبيهات اليوم إلى إجراءات قابلة للإغلاق.
