# 27 — المتابعة اليومية للداخلي

## الهدف

إضافة سجل متابعة يومي لطلاب الداخلي في دور التحفيظ للبنين، بحيث يستطيع المشرف تسجيل الحالة اليومية للطالب من حيث الصلاة والحلقة والحفظ والسلوك والنظافة والنوم والصحة.

## كيانات جديدة

تمت إضافة:

```txt
BoardingCheckIn
```

ويمثل متابعة يومية لطالب داخلي.

## ماذا يسجل؟

- تاريخ المتابعة.
- حضور صلاة الفجر.
- حضور حلقة القرآن.
- صفحات الحفظ.
- صفحات المراجعة.
- درجة السلوك.
- درجة النظافة.
- درجة الانضباط في النوم.
- الحالة الصحية.
- ملاحظة المشرف.
- هل يظهر ملخصها لولي الأمر أم لا.

## Endpoints جديدة

### عرض متابعات طالب داخلي

```http
GET /api/v1/organizations/:organizationId/education/boarding/residents/:residentId/check-ins
```

### إضافة متابعة يومية

```http
POST /api/v1/organizations/:organizationId/education/boarding/residents/:residentId/check-ins
```

مثال:

```json
{
  "fajrPrayer": true,
  "quranSession": true,
  "memorizationPages": 1,
  "revisionPages": 4,
  "behaviorScore": 88,
  "cleanlinessScore": 90,
  "sleepDisciplineScore": 82,
  "healthStatus": "good",
  "supervisorNote": "حضور جيد للحلقة",
  "parentVisible": false
}
```

## تأثير المتابعة اليومية

عند إضافة متابعة يومية، يتم تحديث مؤشرات الطالب الداخلي:

- المتابعة التربوية.
- الإشراف.
- مراجعة القرآن.
- الحالة الصحية.

## تحديث Dashboard

تم تحديث تبويب:

```txt
داخلي البنين
```

ليدعم:

- اختيار طالب داخلي.
- عرض سجل المتابعة اليومية.
- إضافة متابعة يومية.
- تحديد إن كانت المتابعة ظاهرة لولي الأمر أم داخلية فقط.

## سياسة ولي الأمر

المتابعة اليومية قد تحتوي على ملاحظات إشرافية حساسة. لذلك يوجد حقل:

```txt
parentVisible
```

إذا كان `false` فلا تظهر التفاصيل لولي الأمر.

## تحديث Smoke Test

تم تحديث الاختبار ليتحقق من:

- جلب طلاب الداخلي.
- إضافة متابعة يومية.
- التأكد أن المتابعة تظهر في قائمة check-ins.

## المرحلة التالية

- عرض تقويم يومي/أسبوعي للمتابعات.
- تجميع تقرير شهري للداخلي.
- تصنيف الملاحظات: تربوية، صحية، سلوكية، حفظ.
- تنبيهات تلقائية عند انخفاض السلوك أو الحضور أو المراجعة.
- لوحة خاصة للمشرف التربوي.
