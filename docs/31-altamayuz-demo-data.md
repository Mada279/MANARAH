# 31 — بيانات مؤسسة التميز التعليمية

## الهدف

تحديث بيانات النموذج التجريبي في مشكاة لتصبح:

```txt
مؤسسة التميز التعليمية
```

بدل مؤسسة النور التجريبية، مع إظهار دعم مشكاة لإدارة المدارس ودور التحفيظ.

## بيانات المؤسسة

```txt
الاسم: مؤسسة التميز التعليمية
المسؤول: د. مالك بالدي
```

## المدارس

تمت إضافة المدارس التالية كنماذج تشغيل داخل مشكاة:

```txt
مدرسة ١ - لابي
مدرسة ٢ - كنديا
مدرسة ٣ - كوندرا
```

## الأماكن

تم توثيق الأماكن داخل بيانات المؤسسة والفصول:

```txt
لابي
كنديا
كوندرا
القرية
```

## قسم التحفيظ

تم تمثيل قسم التحفيظ بهذه الصورة:

```txt
قسم التحفيظ الخارجي - أولاد
قسم التحفيظ الخارجي - بنات
قسم التحفيظ الداخلي - بنين
```

## قاعدة الداخلي

```txt
الداخلي للبنين فقط.
لا يوجد داخلي للبنات.
البنات لهن قسم خارجي فقط.
ولي الأمر يرى النتائج العامة فقط.
```

## ما تم تحديثه في البيانات

تم تحديث البيانات الافتراضية في:

```txt
server/src/lib/memory-store.ts
```

وشمل ذلك:

- اسم المؤسسة.
- وصف المؤسسة.
- المسؤول.
- البرامج.
- الفصول والحلقات.
- الطلاب.
- الموظفين.
- الماليات.
- الداخلي للبنين.
- بيانات Smoke Test المرتبطة بالمؤسسة.

## ملاحظة عن البيانات القديمة

لأن النظام يستخدم JSON persistence، قد يوجد ملف محلي قديم:

```txt
server/data/manarah-store.json
```

تم تحديث دالة `normalizeStore` بحيث تقوم بترقية بيانات المثال القديمة إلى بيانات مؤسسة التميز التعليمية عند تشغيل السيرفر.

ويمكن أيضًا إعادة التهيئة من لوحة مشكاة أو عبر:

```http
POST /api/v1/dev/reset-store
Authorization: Bearer demo-token
```

## التحقق

تم تشغيل:

```bash
npm run server:check
npm run build
npm run api:smoke
```

وأظهر الاختبار أن المؤسسة أصبحت:

```txt
altamayuz-education
```
