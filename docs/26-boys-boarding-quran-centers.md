# 26 — الداخلي للبنين في دور التحفيظ

## الهدف

إضافة نموذج خاص لدور التحفيظ الداخلية التي تعتمد على مبيت البنين فقط، مع التأكيد أن البنات لا يوجد لهن نظام داخلي في هذا التصور.

هذا القسم يحتاج نظام متابعة مختلف عن إدارة المدرسة العادية، لأنه يشمل:

- الإشراف اليومي.
- المتابعة التربوية.
- التوجيه والسلوك.
- متابعة الورد والحفظ والمراجعة.
- الصحة والانضباط.
- سياسة ما يظهر لولي الأمر.

## قاعدة مهمة

```txt
الداخلي للبنين فقط.
لا يوجد داخلي للبنات.
ولي الأمر يرى النتائج العامة فقط، وليس تفاصيل الإشراف الداخلي.
```

## كيانات جديدة

تمت إضافة:

```txt
BoardingResident
```

ويمثل طالبًا مقيمًا في الداخلي.

يحفظ:

- الطالب.
- الغرفة.
- المشرف التربوي.
- درجة المتابعة التربوية.
- درجة الإشراف.
- درجة مراجعة القرآن.
- الحالة الصحية.
- ملاحظات داخلية.
- هل التقرير العام ظاهر لولي الأمر.

## Endpoints جديدة

### ملخص الداخلي للبنين

```http
GET /api/v1/organizations/:organizationId/education/boarding/summary
```

يرجع:

- عدد الطلاب الداخلي.
- عدد الغرف.
- متوسط التربية.
- متوسط الإشراف.
- متوسط مراجعة القرآن.
- التنبيهات الصحية.
- عدد التقارير الظاهرة لولي الأمر.
- boysOnly = true.
- girlsBoardingSupported = false.

### الطلاب الداخلي

```http
GET /api/v1/organizations/:organizationId/education/boarding/residents
POST /api/v1/organizations/:organizationId/education/boarding/residents
```

### تقرير الداخلي للبنين

```http
GET /api/v1/organizations/:organizationId/education/boarding/reports/summary
```

### نتائج ولي الأمر فقط

```http
GET /api/v1/parent/students/:studentId/results
```

هذا endpoint يراعي أن ولي الأمر يرى النتائج العامة فقط:

- اسم الطالب.
- الفصل/الحلقة.
- التقدم العلمي.
- صفحات الحفظ.
- الحضور.
- حالة المصروفات.
- توصيات عامة.

ولا يرى:

- ملاحظات الإشراف الداخلي التفصيلية.
- ملاحظات السلوك الحساسة.
- تفاصيل المشرفين الداخلية.

## تحديث Dashboard

تمت إضافة تبويب جديد:

```txt
داخلي البنين
```

يعرض:

- عدد الطلاب الداخلي.
- عدد الغرف.
- متوسط التربية.
- متوسط مراجعة القرآن.
- تقرير الداخلي للبنين.
- ملاحظات الإشراف.
- سياسة إظهار النتائج لولي الأمر.
- قائمة طلاب الداخلي.
- إضافة طالب داخلي.

## تحديث Smoke Test

تم تحديث الاختبار ليتحقق من:

- دخول ولي الأمر.
- أن ولي الأمر يرى نتائج الطالب فقط.
- أن تفاصيل الداخلي مخفية عن ولي الأمر.
- أن نظام الداخلي للبنين يعلن صراحة أن البنات غير مدعومات في الداخلي.
- جلب تقرير الداخلي.

## مراحل لاحقة مقترحة

- سجل متابعة يومي لكل طالب داخلي.
- سجل سلوك وتربية منفصل عن تقرير ولي الأمر.
- جدول نوم واستيقاظ وحضور الحلقة.
- متابعة الصحة والنظافة الشخصية.
- زيارات ولي الأمر وتصاريح الخروج.
- صلاحيات دقيقة للمشرف والمربي والمدير.
- تقارير شهرية للداخلي.
