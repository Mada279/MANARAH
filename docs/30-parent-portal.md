# 30 — لوحة ولي الأمر

## الهدف

إضافة واجهة خاصة بولي الأمر لمتابعة نتائج الأبناء فقط، دون عرض تفاصيل الإشراف الداخلي أو الملاحظات الحساسة.

## المسار الجديد

```txt
#parent
```

مثال محلي:

```txt
http://localhost:5173/MANARAH/#parent
```

## تسجيل الدخول

يدخل ولي الأمر من نفس صفحة الدخول:

```txt
#login
```

بالمستخدم التجريبي:

```txt
parent@manarah.local
```

بعد الدخول يتم تحويله تلقائيًا إلى:

```txt
#parent
```

## ماذا يرى ولي الأمر؟

- الأبناء المرتبطون بحسابه فقط.
- التقدم العلمي.
- صفحات الحفظ.
- نسبة الحضور.
- حالة المصروفات.
- توصيات عامة مناسبة للأسرة.
- تأكيد أن تفاصيل الداخلي مخفية.

## ماذا لا يرى ولي الأمر؟

- ملاحظات المشرف الداخلية.
- تفاصيل السلوك الحساسة.
- تفاصيل الإشراف اليومي الدقيقة.
- معلومات الطلاب الآخرين.
- أدوات الإدارة أو الحذف أو التعديل.

## سياسة الخصوصية

تعتمد اللوحة على قاعدة:

```txt
ولي الأمر يتابع النتائج العامة فقط.
الإشراف الداخلي يبقى داخل إدارة الدار.
```

## المرحلة التالية

- دعم أكثر من ابن في الحساب الواحد.
- إشعارات لولي الأمر عند صدور تقرير شهري.
- رسائل مختصرة من الإدارة بدون كشف الملاحظات الداخلية.
- تحميل تقرير PDF مناسب لولي الأمر.
- إقرار استلام ولي الأمر للتقرير.
