# 23 — إدارة المدارس ودور التحفيظ داخل مشكاة

## الهدف

توسيع تصور مشكاة ليظهر بوضوح كنظام إدارة للمدارس ودور التحفيظ، وليس فقط الجمعيات أو البرامج العامة.

هذه المرحلة تضيف طبقة تعليمية داخل Mishkat MVP تشمل:

- إدارة الفصول والحلقات.
- إدارة الطلاب.
- متابعة التقدم العلمي.
- متابعة الحفظ القرآني.
- إدارة المعلمين والموارد البشرية.
- إدارة ماليات المدرسة أو دار التحفيظ.

## ما تم تنفيذه في API

تمت إضافة كيانات جديدة داخل JSON Store:

```txt
EducationClass
Student
StaffMember
FinanceEntry
```

## Endpoints جديدة

### ملخص التعليم والتحفيظ

```http
GET /api/v1/organizations/:organizationId/education/summary
```

يرجع:

- عدد الفصول والحلقات.
- عدد فصول المدرسة.
- عدد حلقات التحفيظ.
- عدد الطلاب.
- عدد الموظفين والمعلمين.
- متوسط التقدم العلمي.
- متوسط صفحات الحفظ.
- نسبة الحضور.
- الإيرادات.
- المصروفات.
- الرصيد.

### الفصول والحلقات

```http
GET  /api/v1/organizations/:organizationId/education/classes
POST /api/v1/organizations/:organizationId/education/classes
```

### الطلاب

```http
GET  /api/v1/organizations/:organizationId/education/students
POST /api/v1/organizations/:organizationId/education/students
```

### الموارد البشرية والمعلمون

```http
GET  /api/v1/organizations/:organizationId/education/staff
POST /api/v1/organizations/:organizationId/education/staff
```

### الماليات

```http
GET  /api/v1/organizations/:organizationId/education/finance
POST /api/v1/organizations/:organizationId/education/finance
```

## ما تم تنفيذه في Dashboard

تمت إضافة تبويب جديد في لوحة مشكاة:

```txt
التعليم والتحفيظ
```

يعرض:

- إجمالي الفصول والحلقات.
- عدد الطلاب.
- عدد المعلمين والموظفين.
- الرصيد المالي.
- متوسط التقدم العلمي.
- متوسط صفحات الحفظ.
- نسبة الحضور.
- قائمة الفصول والحلقات.
- قائمة الطلاب وتقدمهم.
- ملخص الماليات.
- الموارد البشرية.
- إضافة طالب سريعًا.
- إضافة قيد مالي سريعًا.

## أمثلة على الاستخدام

### مدرسة

يمكن استخدام مشكاة لإدارة:

- الصفوف الدراسية.
- الطلاب.
- المعلمين.
- المصروفات الدراسية.
- الحضور.
- التقدم العلمي.
- الموارد البشرية.

### دار تحفيظ

يمكن استخدام مشكاة لإدارة:

- الحلقات.
- المحفظين والمحفظات.
- الطلاب.
- صفحات الحفظ.
- المراجعة والإتقان.
- الحضور.
- الدعم المالي.

### نموذج هجين

يمكن للمؤسسة الجمع بين:

```txt
مدرسة + تحفيظ قرآن
```

من خلال `track = hybrid`.

## تحديث Smoke Test

تم تحديث الاختبار ليتحقق من:

- جلب ملخص التعليم.
- جلب الفصول والحلقات.
- إضافة طالب تجريبي.
- إضافة قيد مالي تجريبي.

## المرحلة التالية

المرحلة القادمة يمكن أن تضيف:

- تعديل وحذف الطلاب والفصول.
- سجلات حضور يومية.
- درجات واختبارات.
- خطط حفظ ومراجعة.
- تقارير ولي الأمر.
- رسوم ومصروفات شهرية.
- رواتب وبدلات وإجازات للموظفين.
- تصدير تقارير تعليمية ومالية.
