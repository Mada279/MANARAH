# 28 — التقرير الشهري للداخلي

## الهدف

إضافة تقرير شهري للداخلي في دور التحفيظ للبنين، لتجميع سجلات المتابعة اليومية وتحويلها إلى مؤشرات قابلة للإدارة والمتابعة.

## Endpoint جديد

```http
GET /api/v1/organizations/:organizationId/education/boarding/reports/monthly
```

ويمكن تحديد الشهر:

```http
GET /api/v1/organizations/:organizationId/education/boarding/reports/monthly?month=2026-07
```

## ماذا يعرض؟

- عدد أيام المتابعة.
- متوسط حضور صلاة الفجر.
- متوسط حضور الحلقة.
- إجمالي صفحات الحفظ.
- إجمالي صفحات المراجعة.
- متوسط السلوك.
- متوسط النظافة.
- متوسط الانضباط في النوم.
- عدد الحالات الصحية أو الإشرافية.
- عدد الملاحظات الداخلية غير الظاهرة لولي الأمر.
- توصيات للمشرف.
- ملخص آمن يمكن عرضه لولي الأمر.

## سياسة ولي الأمر

التقرير الشهري يفرّق بين:

### ما يمكن عرضه لولي الأمر

- عدد أيام المتابعة.
- حضور الحلقة.
- حضور الفجر كنسبة عامة.
- إجمالي الحفظ والمراجعة.
- توصيات عامة.

### ما يبقى داخليًا

- ملاحظات المشرف التفصيلية.
- تفاصيل السلوك الحساسة.
- ملاحظات الصحة والإشراف الدقيقة.
- تقييمات داخلية لا تخص ولي الأمر مباشرة.

- تصدير التقرير الشهري PDF.
- تقرير شهري لكل طالب داخلي منفصل.
- مقارنة شهرية بين الطلاب والغرف.
- تنبيهات تلقائية عند انخفاض الفجر أو الحلقة أو السلوك.
- لوحة خاصة للمشرف التربوي اليومي.
