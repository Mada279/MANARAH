# 02 — منظومة منارة

## الخريطة العامة

```txt
MANARAH
│
├── Meraj | معراج
│   رحلة الإنسان
│
├── Mishkat | مشكاة
│   تشغيل المؤسسات
│
├── Impact Engine | محرك الأثر
│   قياس النتائج وتحويل البيانات إلى قرارات
│
├── Digital Waqf | الوقف الرقمي
│   الاستدامة والتمويل طويل الأجل
│
├── Knowledge + Academy
│   المعرفة والتعليم والتوثيق
│
├── Community
│   المجموعات والمبادرات والتطوع
│
├── Developer Portal
│   API و SDK والتكاملات
│
└── Manarah OS
    النواة المشتركة لكل المنتجات
```

## فلسفة الترابط

لا يتم بناء كل منتج كجزيرة مستقلة. كل منتج يستخدم خدمات Manarah OS المشتركة:

- الهوية.
- المؤسسات.
- الصلاحيات.
- النماذج.
- سير العمل.
- المستندات.
- الإشعارات.
- التقارير.
- الأثر.
- التكاملات.

## رحلة الإنسان

```txt
ينضم المستخدم
  ↓
يبني ملفه الشخصي
  ↓
يتعلم وينمو
  ↓
يتطوع ويشارك
  ↓
يدعم أو يستفيد
  ↓
يرتبط بمؤسسة أو مبادرة
  ↓
يتحول نشاطه إلى أثر قابل للقياس
```

## رحلة المؤسسة

```txt
تسجل المؤسسة
  ↓
تنشئ الفرق والصلاحيات
  ↓
تدير البرامج والمستفيدين
  ↓
تدير المتطوعين والمستندات
  ↓
تقيس المؤشرات
  ↓
تصدر تقارير أثر
  ↓
تحصل على توصيات وتحسينات
```

## رحلة الأثر

```txt
بيانات تشغيلية
  ↓
تنظيف وتنظيم
  ↓
مؤشرات أداء
  ↓
تحليل وتوصيات
  ↓
تقارير وقصص نجاح
  ↓
قرارات أفضل
```

## رحلة الوقف

```txt
أصل وقفي أو مورد مستدام
  ↓
حوكمة ومصارف
  ↓
تمويل برامج ومشاريع
  ↓
قياس أثر التمويل
  ↓
تقارير شفافة للواقفين والشركاء
```

## الأولوية

رغم أن المنظومة واسعة، إلا أن التنفيذ يجب أن يبدأ بالحد الأدنى القابل للاستخدام:

```txt
Manarah OS Core
  +
Mishkat MVP
  +
Impact Dashboard
```
