# 32 — اللغات الثلاث والوضع الفاتح/الداكن وERP المدرسة

## الهدف

إضافة أساس عملي لدعم ثلاث لغات ووضع فاتح/داكن، مع إظهار مشكاة كنظام ERP لإدارة المدارس ودور التحفيظ.

## اللغات المدعومة

تمت إضافة أساس i18n للغات:

```txt
العربية
English
Français
```

الملفات:

```txt
src/lib/preferences.tsx
```

ويحفظ اختيار اللغة في:

```txt
localStorage.manarah_language
```

## الوضع الفاتح والداكن

تمت إضافة وضعين:

```txt
dark
light
```

ويحفظ الاختيار في:

```txt
localStorage.manarah_theme
```

كما تم تحديث CSS لدعم الوضع الفاتح على مستوى البطاقات والنصوص والحقول.

## أين تظهر أدوات اللغة والثيم؟

- صفحة تسجيل الدخول.
- لوحة مشكاة.
- لوحة ولي الأمر.

## ERP المدرسة

تمت إضافة تبويب جديد في لوحة مشكاة:

```txt
ERP المدرسة
```

يعرض:

- الإدارة المالية والمصروفات.
- إدارة الطلاب.
- إدارة المدرسين والموارد البشرية.
- إضافة قيد مالي.
- إضافة مدرس/موظف.
- الإيرادات.
- المصروفات.
- رواتب المدرسين والموظفين.

## ملاحظة مهمة

الدعم اللغوي الحالي هو أساس قابل للتوسع، وتمت ترجمة عناصر الواجهة الأساسية والتنقلات. أما بيانات المؤسسة نفسها مثل أسماء المدارس والأماكن فهي بيانات عربية لأنها تخص مؤسسة التميز التعليمية.

## المرحلة التالية

- نقل كل النصوص الثابتة إلى ملفات ترجمة كاملة.
- إضافة ترجمة كاملة للموقع التسويقي.
- إضافة إعدادات لغة لكل مستخدم.
- توسيع ERP ليشمل الجداول التفصيلية والتحصيل والفواتير والرواتب.
