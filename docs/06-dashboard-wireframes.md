# 06 — شاشات لوحة التحكم

هذه الوثيقة تصف شاشات Mishkat MVP نصيًا قبل التصميم والتنفيذ.

## 1. تسجيل الدخول

العناصر:

- شعار منارة.
- البريد الإلكتروني.
- كلمة المرور.
- زر دخول.
- رابط نسيت كلمة المرور لاحقًا.

## 2. اختيار المؤسسة

إذا كان المستخدم عضوًا في أكثر من مؤسسة:

- قائمة المؤسسات.
- زر دخول للمؤسسة.
- زر إنشاء مؤسسة جديدة إذا كان مصرحًا.

## 3. لوحة المؤسسة

أهم شاشة في MVP.

تحتوي على:

- عدد البرامج النشطة.
- عدد المستفيدين.
- عدد المتطوعين.
- ساعات التطوع.
- مؤشرات أثر رئيسية.
- آخر الأنشطة.
- تنبيهات تحتاج إجراء.

تخطيط مقترح:

```txt
[عنوان المؤسسة]                         [فترة التقرير]

[البرامج] [المستفيدون] [المتطوعون] [ساعات التطوع]

[رسم مؤشرات الأثر]       [آخر الأنشطة]

[البرامج النشطة]         [تنبيهات]
```

## 4. البرامج

قائمة البرامج:

- اسم البرنامج.
- الحالة.
- المدير.
- عدد المستفيدين.
- عدد المتطوعين.
- نسبة التقدم.
- آخر تحديث.

إجراءات:

- إنشاء برنامج.
- تعديل.
- أرشفة.
- فتح التفاصيل.

## 5. تفاصيل البرنامج

تبويبات:

- نظرة عامة.
- المستفيدون.
- المتطوعون.
- المؤشرات.
- المستندات.
- التقارير.

## 6. المستفيدون

جدول:

- الاسم.
- البرنامج.
- الفئة العمرية.
- المدينة.
- الحالة.
- تاريخ الإضافة.

إجراءات:

- إضافة مستفيد.
- استيراد CSV لاحقًا.
- تصفية حسب البرنامج.

## 7. المتطوعون

جدول:

- الاسم.
- المهارات.
- البرنامج.
- إجمالي الساعات.
- الحالة.

إجراءات:

- إضافة متطوع.
- تحديث ساعات التطوع.
- ربط ببرنامج.

## 8. المستندات

- قائمة الملفات.
- التصنيف.
- البرنامج المرتبط.
- من رفع الملف.
- تاريخ الرفع.

## 9. مؤشرات الأثر

- تعريف مؤشر.
- تحديد الهدف.
- إدخال قيمة دورية.
- عرض تقدم المؤشر.

مثال:

```txt
مؤشر: عدد المستفيدين
الهدف: 1000
الحالي: 720
التقدم: 72%
```

## 10. التقارير

- اختيار برنامج.
- اختيار فترة.
- توليد تقرير.
- عرض ملخص.
- تصدير لاحقًا.

## 11. الإعدادات والأعضاء

- معلومات المؤسسة.
- الأعضاء.
- الأدوار.
- الصلاحيات الأولية.

## الأولوية في التنفيذ

الأسبوع الأول:

- تسجيل الدخول.
- لوحة المؤسسة.
- البرامج.

الأسبوع الثاني:

- المستفيدون.
- المتطوعون.

الأسبوع الثالث:

- مؤشرات الأثر.
- التقارير الأولية.

الأسبوع الرابع:

- المستندات.
- الإعدادات.
- تحسينات UX.
