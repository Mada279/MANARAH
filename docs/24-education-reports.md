# 24 — تقارير التعليم والماليات

## الهدف

إضافة تقرير أولي داخل مشكاة للمدارس ودور التحفيظ، بحيث لا تقتصر اللوحة على عرض البيانات، بل تترجمها إلى ملخص تنفيذي وتوصيات.

## Endpoint جديد

```http
GET /api/v1/organizations/:organizationId/education/reports/summary
```

## ماذا يرجع التقرير؟

يرجع التقرير:

- عنوان التقرير.
- تاريخ التوليد.
- ملخص التعليم والتحفيظ.
- أبرز النتائج.
- المخاطر والتنبيهات.
- التوصيات.
- الملاحظات المالية.

## أمثلة على أبرز النتائج

- عدد الفصول والحلقات.
- عدد حلقات التحفيظ.
- متوسط التقدم العلمي.
- متوسط صفحات الحفظ.
- نسبة الحضور.
- الرصيد المالي.

## أمثلة على المخاطر

- طلاب منخفضو الحضور.
- طلاب لديهم مستحقات غير مسددة.
- رصيد مالي سالب.

## أمثلة على التوصيات

- إنشاء تقرير شهري للتقدم العلمي والحفظ.
- ربط الطلاب منخفضي الحضور بخطة متابعة مع ولي الأمر.
- تخصيص مؤشرات منفصلة للدراسة والحفظ والمراجعة والحضور.
- مراجعة النماذج الناجحة في الحلقات أو الفصول ذات الأداء المرتفع.

## التحديث في Dashboard

تم تحديث تبويب:

```txt
التعليم والتحفيظ
```

ليعرض بطاقة:

```txt
تقرير التعليم والتحفيظ المختصر
```

وتشمل:

- أبرز النتائج.
- التوصيات.
- التنبيهات.
- الملاحظات المالية.

## تحديث Smoke Test

تم تحديث الاختبار ليتحقق من:

```txt
GET /education/summary
GET /education/classes
POST /education/students
POST /education/finance
GET /audit-logs
```

## المرحلة التالية

يمكن لاحقًا إضافة تقارير أكثر تخصصًا:

```txt
تقرير أداء الطالب
تقرير حلقة التحفيظ
تقرير الفصل الدراسي
تقرير الحضور والغياب
تقرير الرسوم والمستحقات
تقرير الرواتب والموارد البشرية
تقرير المصروفات والإيرادات
تقرير شامل PDF
```
