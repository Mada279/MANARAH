# MANARAH
Building the Digital Infrastructure for Human Development  مشروع منارة (Manarah) الرؤية (Vision)  أن تصبح منارة المرجع الرقمي الإسلامي الأول عالميًا، بحيث يجد المسلم داخلها كل ما يحتاجه في حياته اليومية من مصدر موثوق واحد.  وليست مجرد تطبيق إسلامي.  بل:  النظام التشغيلي الرقمي للمسلم (Digital Operating System for Muslims).
